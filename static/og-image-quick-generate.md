# Quick OG Image Generation

## Fastest Method (2 minutes)

1. **Open the HTML template**:
   - Navigate to: `static/og-image-generator.html`
   - Open in Chrome/Edge browser

2. **Take screenshot**:
   - Press `F12` (Developer Tools)
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
   - Type: `screenshot`
   - Select: **"Capture screenshot"**
   - Save as: `og-image.jpg` in the `static/` folder

3. **Verify dimensions**:
   - The image should be exactly **1200x630 pixels**
   - If different, resize using any image editor

## Alternative: Browser Extension

1. Install "Full Page Screen Capture" extension
2. Open `og-image-generator.html`
3. Capture the page
4. Crop to 1200x630px
5. Save as `og-image.jpg`
